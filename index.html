Help Center

🧠 How does it detect duplicates?

Unlike basic file scanners that check exact file sizes, EchoScan uses a deep learning neural network (ResNet50). It actually "looks" at the visual features of your media, allowing it to find duplicates even if they are renamed, cropped, or recolored.

🎚️ What is the 'Threshold' slider?

The similarity threshold defines how strict the AI should be. At 99%, only exact, pixel-perfect images match. At 80%, the AI will group images together that merely look similar (like taking a burst of photos).

🗑️ Is Auto-Remove safe?

Yes! Auto-remove permanently deletes the trailing redundant copies in any group the AI flags, but it always keeps the main original file completely safe untouched.
